# Division Auto Body Inc Website

This repository contains the GitHub Pages website for Division Auto Body Inc in Keyport, NJ.

## Overview

Division Auto Body Inc has served Keyport since 1977. The site highlights the shop’s collision repair services, premium finish work, fleet services, insurance support, and proof-of-work gallery.

## Pages

- `index.html` — homepage
- `services.html` — services overview
- `gallery.html` — proof of work / before-and-after gallery
- `about.html` — company background
- `contact.html` — contact details and call to action
